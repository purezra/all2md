# All2MD Chrome Plugin

Chrome 插件版面向更稳定的文章页面提取场景，适合作为当前项目的内测分发版本。

当前稳定支持：

- 微信公众号文章
- 知乎回答 / 文章
- 今日头条文章
- GitHub 仓库 README

当前能力：

- 提取当前页面为 Markdown
- 本地预览 Markdown
- 下载 `.md`
- 下载 `.zip`，包含 Markdown 和图片
- 在浏览器本地完成提取，不上传正文到远端服务

## 本地加载

1. 打开 `chrome://extensions/`
2. 开启右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前目录下的 `plugin/`

## 使用方式

1. 打开一个支持的平台页面
2. 点击浏览器工具栏里的 `All2MD`
3. 点击“提取当前页”
4. 复制 Markdown，或下载 `.md` / `.zip`

## 分发建议

如果暂时不上架 Chrome 商店，推荐把整个 `plugin/` 目录压缩成 zip，发给测试用户，让他们以“加载已解压的扩展程序”的方式安装。

## 现阶段不主推的能力

- X / Twitter 页面提取暂不作为插件稳定能力对外宣传
- 个别页面的图片仍可能受防盗链影响，导致 ZIP 中少量图片保留远程地址
