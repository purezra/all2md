# All2MD Chrome Plugin

这是一个可直接加载到 Chrome 的 MVP 插件，目标是把“当前打开的页面”提取成 Markdown。

当前优先支持：

- `x.com` / `twitter.com`
- `github.com` 仓库 README 页
- 微信公众号文章页
- 知乎文章 / 回答页
- 今日头条文章页

## 本地加载

1. 打开 Chrome 扩展页：`chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前目录下的 `plugin/`

## 使用方式

1. 打开一个支持的平台页面
2. 点击浏览器工具栏里的 `All2MD Current Page`
3. 点击“提取当前页”
4. 复制 Markdown 或下载 `.md`

## 说明

- 这个插件走的是“当前页面 DOM 提取”，会复用浏览器里已经加载好的内容
- 对 `X` 这类强依赖登录态和客户端渲染的平台，插件通常比服务端抓取更稳
- 目前是无构建的原型版本，后续可以再和主项目抽共享逻辑
