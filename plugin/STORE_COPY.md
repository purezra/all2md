# Chrome Web Store Copy

## Short Description

Convert the current article page into clean Markdown with preview and ZIP export.

## Detailed Description

All2MD helps you turn supported article pages into editable Markdown directly inside Chrome.

Current stable support:

- WeChat public articles
- Zhihu answers and articles
- Toutiao article pages
- GitHub repository README pages

Key features:

- Extract the current page into Markdown
- Preview Markdown before exporting
- Copy Markdown to clipboard
- Download `.md`
- Download `.zip` with images
- Run extraction locally in the browser

Typical use cases:

- Save articles into Obsidian or Notion workflows
- Archive reference pages as Markdown
- Export GitHub README content for editing
- Keep a local offline package with text and images

## 中文详细介绍

All2MD 可以直接在 Chrome 中把当前文章页整理成可编辑的 Markdown。

当前稳定支持：

- 微信公众号文章
- 知乎回答与文章
- 今日头条文章页
- GitHub 仓库 README 页面

主要功能：

- 当前页一键提取 Markdown
- 提取结果本地预览
- 一键复制 Markdown
- 下载 `.md`
- 下载包含图片的 `.zip`
- 在浏览器本地完成提取

适合用于：

- 保存文章到 Obsidian、Typora、VS Code
- 留存文章资料和知识库素材
- 提取 GitHub README 作为编辑底稿
- 打包离线阅读内容
